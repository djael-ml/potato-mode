# Potato Mode Extension

Une extension radicale pour transformer n'importe quel site web en version "Potato". Optimisée pour les performances extrêmes.

## Fonctionnalités
- ⚙️ **Modularité Totale** : Activez ou désactivez chaque optimisation individuellement.
- 🚫 **Désactivation des animations** : Supprime les transitions et animations gourmandes.
- 🎞️ **Blocage des GIFs** : Stopper les images animées distrayantes.
- 📹 **Mise en pause vidéo** : Bloque l'autoplay et force la pause sur les lecteurs media.
- 🖼️ **Blocage des Images** (Optionnel).
- 🎨 **Design Plat** : Supprime les ombres et les bordures arrondies pour un rendu minimaliste.
- 🌙 **Force Dark Mode** (Optionnel, ultra léger).
- 🔡 **Polices Systèmes** uniquement.

## Confidentialité
Potato Mode respecte votre vie privée. Aucune donnée n'est collectée ou transmise à des tiers. Plus de détails dans [PRIVACY.md](./PRIVACY.md).

## Installation (Firefox)
1. Téléchargez ou clonez ce dépôt.
2. Ouvrez Firefox et accédez à `about:debugging#/runtime/this-firefox`.
3. Cliquez sur **"Load Temporary Add-on..."**.
4. Sélectionnez le fichier `manifest.json`.

## Performance
L'extension utilise des sélecteurs CSS optimisés et un `MutationObserver` économe en ressources pour garantir un impact minimal sur le CPU.
